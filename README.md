# thyroid-ml-prediction

# Predicting Thyroid Dysfunction Using Machine Learning

## Overview

This project investigates whether routinely collected clinical and metabolic biomarkers can be used to predict thyroid dysfunction using data from the National Health and Nutrition Examination Survey (NHANES).

The long-term objective is to develop an interpretable machine learning model that can assist in the early screening of thyroid disorders using commonly available laboratory and demographic information.

---

## Research Objectives

- Investigate relationships between thyroid biomarkers and metabolic health
- Identify significant predictors of thyroid dysfunction
- Develop baseline and advanced machine learning models
- Compare statistical and machine learning approaches
- Evaluate clinically meaningful features for early detection

---

## Dataset

NHANES 2007–2008

Current datasets merged:

- DEMO_E
- BMX_E
- GLU_E
- GHB_E
- THYROD_E

Current sample size:

- 2,953 participants
